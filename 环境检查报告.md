## 安装 VS Build Tools

### 方法 1: 运行安装脚本（推荐）

```powershell
# 以管理员身份运行 PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.\install_vs_buildtools.ps1
```

### 方法 2: 手动安装

1. 访问: https://visualstudio.microsoft.com/visual-cpp-build-tools/
2. 下载并运行安装程序
3. 选择 **"使用 C++ 的桌面开发"** 工作负载
4. 点击安装（约 6-7GB，需要 10-20 分钟）

---

## 安装完成后验证

```powershell
# 验证 MSVC 编译器
where cl.exe

# 应输出类似:
# C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\VC\Tools\MSVC\...\bin\Hostx64\x64\cl.exe

# 测试 Rust 编译
cargo new test_build
cd test_build
cargo build
```

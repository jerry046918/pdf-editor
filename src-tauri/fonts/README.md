# 内置字体

本目录存放 PDF 编辑器使用的内置字体。

## 当前字体

| 字体 | 文件 | 用途 |
|------|------|------|
| 思源黑体 | NotoSansSC-Regular.ttf, NotoSansSC-Bold.ttf | 黑体、微软雅黑、等线 |
| 思源宋体 | NotoSerifSC-Regular.ttf, NotoSerifSC-Bold.ttf | 宋体、仿宋 |
| 全字库楷体 | TW-Kai-98_1.ttf | 楷体 |

## 字体映射

| 原字体 | 内置替代 |
|--------|----------|
| 黑体/SimHei | NotoSansSC-Bold |
| 宋体/SimSun | NotoSerifSC-Regular |
| 楷体/KaiTi | TW-Kai-98_1 |
| 微软雅黑 | NotoSansSC-Regular |
| 等线 | NotoSansSC-Regular |
| 仿宋 | NotoSerifSC-Regular |

## 字体来源

- **思源黑体/宋体**: [Google Fonts](https://fonts.google.com/noto) - SIL Open Font License
- **全字库楷体**: [台湾全字库](https://data.gov.tw/dataset/5961) - 开放使用

## 注意事项

- 字体文件会增加应用安装包大小（约 120MB）
- 如果此目录缺少字体文件，应用会回退使用系统字体或内置 china-s
